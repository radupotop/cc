Credit Cards comparison tool
============================

For the Romanian market


Best cards for
--------------

* cashback
* shopping
* students
* travel (wizz) 
* long-term loan

Filter by
---------

* Lowest AER / interest free loan
* Easiest to get

* Car loan

Wizard
------
